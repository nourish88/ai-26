# Gereksinimler

- Python 3.12
- Kök dizinde .env dosyası bulunmalı (bkz. .env-sample)

# Kurulum

1. Python 3.12 kurulur.
2. Virtual environment kurulur.

```python3 -m venv .venv```

3. Virtual environment aktivasyonu yapılır.

```source .venv/bin/activate```

4. Uygulamanın bağımlı olduğu kütüphaneler kurulur.

```pip install -r requirements.txt```
